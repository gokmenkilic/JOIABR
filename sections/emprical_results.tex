\section{Empirical Results}

This section presents the empirical results of the machine learning and deep learning models used for stock price prediction and portfolio optimisation. The results are reported in three stages. First, the main hyperparameter-tuning results are presented for the LSTM and XGBoost models. Second, the best-performing configuration of each model is compared using prediction error metrics. Third, the sequence-based models are evaluated in the portfolio optimisation stage.

\subsection{Hyperparameter tuning results}

Bold values in Table~\ref{tab:tuning-results} indicate the best-performing configuration within each model and error metric group. Since MAE, MSE and RMSE are error metrics, lower values indicate better predictive performance.
